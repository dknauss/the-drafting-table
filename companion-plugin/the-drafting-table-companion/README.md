# The Drafting Table Companion

Optional companion plugin for the The Drafting Table theme.

## What it adds

- Demo content onboarding and removal controls in `Appearance > Themes`.
- Optional SEO/meta helpers (meta description, JSON-LD, paged archive noindex).

## Notes

- This plugin is intentionally excluded from the WordPress.org directory build profile for the theme.
- It is activated automatically in local `wp-env` test setup.
